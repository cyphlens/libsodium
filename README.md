## Installation

libsodium is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
  pod 'libsodium', :git => 'https://github.com/alexinx/libsodium.git', :tag => '1.0.20'
```
